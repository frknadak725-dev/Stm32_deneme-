# Stm32_deneme-
# STM32 HAL Examples

This repository contains small **STM32 embedded system examples** created while learning microcontroller programming using **HAL libraries** and
 **STM32CubeMX**.

The goal of this repository is to build a personal reference library of commonly used peripherals and configurations that can be reused
 in future embedded projects.

Each folder contains a minimal example demonstrating a specific peripheral or concept.

---

## Development Environment

* Microcontroller: STM32 (Discovery board)
* Configuration Tool: STM32CubeMX
* IDE: Keil MDK-ARM
* Language: C
* Library: STM32 HAL

---

## Example Topics

* GPIO Blink
* GPIO Input
* External Interrupt (EXTI)
* Timers
* ADC
* PWM
* UART Communication

Each example includes a simple `main.c` implementation and a short explanation of the configuration and functionality.

---

## Purpose

This repository serves as a **learning archive and quick reference** for STM32 peripheral usage.
The examples are intentionally minimal and designed to demonstrate the core concept of each peripheral.

These snippets can be reused and adapted for future embedded system projects.
